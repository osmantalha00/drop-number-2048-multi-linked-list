# 🎮 Drop Number (2048) - Multi Linked List

Java-based implementation of the Drop Number (2048-like) game using a **custom multi-linked list** instead of arrays.

## 🚀 Features

* 7x5 grid game
* Downward tile movement
* Tile merging (2+2=4, etc.)
* Java Swing GUI
* Object-oriented design

## 🧠 Purpose

Demonstrates game logic and data structure implementation without using arrays.
